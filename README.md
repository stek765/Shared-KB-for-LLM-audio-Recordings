# Shared KB with LLM

Knowledge base personale per appunti universitari, generata automaticamente da registrazioni audio e interrogabile via LLM.

Le note vengono prodotte da [StudyMate](https://github.com/stek765/studymate) e sincronizzate qui tramite Obsidian Git. Ogni cartella corrisponde a una materia; ogni file `.md` è una lezione con wikilink, diagrammi Mermaid e frontmatter YAML.

La KB include anche un layer RAG: le note vengono indicizzate vettorialmente e interrogabili in linguaggio naturale tramite Claude.
